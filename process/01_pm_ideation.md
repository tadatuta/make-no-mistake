# Шаг 1: Генерация и выбор идеи стартапа (Product Manager)
**Роль**: Product Manager (PM)
**Отчет для**: CEO (Antigravity)

## 1. Вводные данные и ограничения
* **Бюджет**: 0$ (Без инвестиций).
* **Создание**: Полностью с помощью AI.
* **Маркетинг**: Бесплатные каналы (Product Hunt, SEO, виральность, соцсети).
* **Инфраструктура**: Бесплатные тарифы (Vercel/Netlify, Supabase/Firebase/GitHub, Free-tier AI API или Bring-Your-Own-Key).

## 2. Генерация идей (Штурм)

### Идея 1: Telegram-бот "AI Учитель Языков"
* **Описание**: Бот для языковой практики через ролевые игры.
* **Плюсы**: Высокий retention, удобная платформа (Telegram).
* **Минусы**: Сложно монетизировать без затрат на LLM API (free tier быстро закончится при масштабировании). 

### Идея 2: Web-приложение "Bring-Your-Own-Key AI Content Repurposer"
* **Описание**: Инструмент для контент-мейкеров. Пользователь вставляет ссылку на YouTube-видео или длинную статью, вводит свой бесплатный API-ключ Gemini/OpenAI, и приложение генерирует тред для X (Twitter), пост для LinkedIn и SEO-статью.
* **Плюсы**: 0$ затрат на server-side генерацию. Полностью фронтенд-решение или легкий бэкенд на Vercel (Next.js). Легко вирусится среди криэйторов.
* **Минусы**: UX с вводом своего ключа может отпугнуть часть аудитории (но целевая аудитория инди-хакеров и криэйторов к этому привыкла).

### Идея 3: "Aesthetic Markdown Portfolio Generator"
* **Описание**: Пользователь пишет о себе в простом Markdown, а приложение генерирует красивый, современный сайт-портфолио, который можно в 1 клик задеплоить на GitHub Pages.
* **Плюсы**: Вообще не нужен AI для генерации в рантайме, 100% бесплатно.
* **Минусы**: Меньше "AI-магии" в самом продукте, сложнее продавать как инновацию.

## 3. Выбор и Утверждение Идеи
**Решение PM**: Выбираем **Идею 2** (AI Content Repurposer с BYOK - Bring Your Own Key).
**Рабочее название**: *ContentAlchemist* (или *RepurposeAI*).

**Почему это лучший выбор:**
1. **Решает реальную боль**: Авторы тратят часы на дистрибуцию контента по разным платформам.
2. **0$ костов при масштабировании**: Вся тяжелая ИИ-работа оплачивается самими пользователями через их API ключи (у Gemini есть отличный бесплатный лимит, который мы будем рекомендовать).
3. **Бесплатный хостинг**: Приложение будет статичным SPA или SSR на Next.js (Vercel Free Tier).
4. **Виральный потенциал**: В конце каждого сгенерированного поста можно добавлять (опционально) вотермарку "Сгенерировано через RepurposeAI", создавая виральный цикл.

## 4. MVP Scope (Для передачи Дизайнеру и Разработчику)
* **Страница Landing/App**: Одностраничное приложение.
* **Блок настроек**: Поле для ввода Gemini API Key (сохраняется локально в localStorage).
* **Блок ввода**: Поле для текста или URL (для извлечения текста).
* **Блок вывода**: Три колонки или вкладки (Twitter, LinkedIn, Blog Post) с готовым текстом и кнопкой "Copy".
* **Логика**: Промпты "под капотом", которые трансформируют исходный текст в нужные форматы с учетом лучших практик соцсетей.

## 5. Рефлексия (Выводы из первого шага)
* **Инсайт**: Ключ к стартапу с 0 инвестиций в эпоху AI — перенос вычислительных затрат на клиента (клиентский рендеринг и Bring-Your-Own-Key для API).
* **Проблема**: Как убедить пользователя ввести ключ?
* **Решение**: Добавить четкую инструкцию со скриншотами "Как получить бесплатный Gemini API ключ за 1 минуту" и обеспечить безопасность (ключ хранится только в браузере `localStorage`).
