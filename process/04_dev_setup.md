# Шаг 4: Разработка MVP (Developer Setup)
**Роль**: Developer
**Проект**: RepurposeAI

## 1. Инициализация проекта
* Создан проект на базе React + TypeScript с помощью Vite (`npx create-vite@latest repurpose-ai --template react-ts`).
* Выбор Vite обоснован максимальной скоростью разработки и сборки, а также нулевой стоимостью статического хостинга.

## 2. Установка зависимостей
* Установлен `tailwindcss`, `postcss` и `autoprefixer` для быстрой и гибкой стилизации (utility-first подход).
* Инициализирован конфиг Tailwind (`npx tailwindcss init -p`).
* Установлен пакет `lucide-react` для использования красивых и легких SVG-иконок.

## 3. Базовая архитектура
В директории `src/` будет следующая структура:
* `components/` - Переиспользуемые UI компоненты (кнопки, инпуты).
* `hooks/` - Кастомные хуки (например, `useGemini`, `useLocalStorage`).
* `utils/` - Вспомогательные функции (промпты, парсинг текста).
* `App.tsx` - Главный компонент, связывающий UI и логику.

## 4. Рефлексия (Выводы из четвертого шага)
* **Инсайт**: Использование готовых инструментов сборки (Vite) и стилизации (Tailwind) экономит десятки часов. Для AI-стартапа с бюджетом $0 скорость итераций — главное конкурентное преимущество.
* **План действий**: Далее необходимо настроить конфигурацию Tailwind, добавить базовые стили в `index.css` и приступить к верстке MVP по спецификациям Дизайнера.
